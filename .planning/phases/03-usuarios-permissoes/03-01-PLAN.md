       - `update(UpdateUserRequest $request, User $user)`: Update user attributes, sync roles if provided (using `sync()` on the `roles` relationship), load fresh roles, return 200. Note: `updated_by` should be set to the authenticated user's ID. Use `$request->user()->id`.

    **Routes** (`routes/api.php`):

    Route::middleware('auth:sanctum')->group(function () {
        Route::apiResource('users', UserController::class);
        Route::apiResource('roles', RoleController::class);

    **Note on `hasPermission`:** Add this method to User model if not done in Task 1:
    ```php
    public function hasPermission(string $slug): bool
    {
        return $this->roles()
            ->whereHas('permissions', fn($q) => $q->where('slug', $slug))
            ->exists();
    }
    ```
    This queries DB directly and works without eager-loading (avoids N+1 for single checks). For route model binding policy checks, the roles will be loaded when needed.
